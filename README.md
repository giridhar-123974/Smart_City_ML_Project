# Smart City Machine Learning Project

## Project Overview
This project applies Machine Learning techniques to Smart City data
to improve urban decision making. The implemented module predicts
Air Quality (CO levels) using sensor data.

## Dataset
- Air Quality Dataset
- Sensor-based air pollution data
- Target variable: CO(GT)

## Machine Learning Model
- Random Forest Regressor

## Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)

## Results
- Feature importance graph and CSV
- Actual vs Predicted CO graph
- Trained model saved as .pkl file

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Folder Structure
- datasets/ : datasets used
- notebooks/ : ML notebooks
- models/ : trained models
- results/ : output graphs
- report/ : final report
